1. 텍스트 생성형 AI
텍스트 데이터를 학습하여 새로운 글을 생성하는 AI입니다. 이 AI는 대화형 챗봇, 소설 생성, 기사 작성 등에 활용됩니다.

예시: GPT (Generative Pre-trained Transformer)
OpenAI에서 개발한 GPT는 대규모 언어 모델로, 주어진 텍스트 데이터를 학습한 후, 새로운 텍스트를 생성하는 AI입니다. 다양한 주제에 대해 자연스러운 대화나 글을 생성할 수 있으며, 사람처럼 질문에 답하거나 문서를 작성하는 데 매우 유능합니다.

BERT (Bidirectional Encoder Representations from Transformers)
BERT는 문장 내의 문맥을 양방향으로 이해하는 언어 모델입니다. 텍스트를 생성하기보다는 텍스트의 의미를 이해하고 분류하는 작업에 더 적합합니다.

2. 이미지 생성형 AI
이미지 데이터를 학습하여 새로운 이미지를 생성하는 AI로, 디지털 아트나 광고, 게임 디자인 등에서 활용됩니다.

예시: DALL·E
OpenAI에서 개발한 DALL·E는 텍스트 설명을 기반으로 이미지를 생성하는 모델입니다. 사용자가 입력한 설명을 바탕으로 독창적이고 창의적인 이미지를 생성할 수 있습니다.

MidJourney
MidJourney는 예술적 스타일과 독특한 디자인의 이미지를 생성하는 AI로, 디지털 아트 분야에서 주로 사용됩니다. 사용자가 원하는 이미지 스타일과 구성을 지정할 수 있으며, 매우 독창적이고 세련된 결과물을 제공합니다.

Stable Diffusion
Stable Diffusion은 이미지 생성을 위한 오픈소스 모델로, 텍스트 기반의 이미지 생성뿐만 아니라 기존 이미지를 변형하거나 다양한 스타일로 변환하는 데도 사용됩니다.

3. 음악 생성형 AI
음악 데이터를 학습하여 새로운 곡을 작곡하는 AI입니다. 주로 미디어 콘텐츠, 영화, 게임 배경음악 등에 활용됩니다.

예시: Jukedeck
Jukedeck은 AI가 자동으로 음악을 작곡해 주는 서비스로, 사용자가 원하는 스타일이나 분위기를 선택하면 AI가 그에 맞는 음악을 생성합니다.

OpenAI MuseNet
MuseNet은 다양한 악기와 음악 스타일을 학습하여 클래식부터 팝까지 다양한 장르의 음악을 생성할 수 있는 AI입니다.

4. 비디오 생성형 AI
비디오 데이터를 학습하여 새로운 비디오 콘텐츠를 생성하거나 기존 비디오를 변형하는 AI입니다.

예시: Synthesia
Synthesia는 텍스트 입력만으로 AI가 비디오를 생성하는 플랫폼으로, 사용자가 원하는 메시지를 전달하는 비디오를 빠르게 만들 수 있습니다. 주로 교육 콘텐츠나 기업 홍보 영상 제작에 사용됩니다.

Runway ML
Runway ML은 이미지와 비디오 데이터를 활용해 AI가 새로운 영상을 만들거나 기존 영상을 편집하는 데 사용되는 도구입니다. 비주얼 아티스트나 크리에이터들이 주로 사용합니다.

5. 음성 생성형 AI
음성 데이터를 학습하여 새로운 음성을 생성하는 AI로, 텍스트를 음성으로 변환하는 TTS(Text-to-Speech) 기술에 사용되며, 자동 응답 시스템이나 오디오북, 방송 등에서 활용됩니다.

예시: Google DeepMind’s WaveNet
WaveNet은 사람의 목소리처럼 자연스럽고 고품질의 음성을 생성하는 AI입니다. 텍스트를 입력하면 해당 텍스트를 자연스럽게 읽어주는 음성을 생성할 수 있습니다.

Amazon Polly
Amazon Polly는 텍스트를 음성으로 변환해 주는 서비스로, 여러 언어와 다양한 목소리 톤을 제공하여 사용자 맞춤형 음성을 생성할 수 있습니다.

6. 코드 생성형 AI
코드 데이터를 학습하여 프로그래밍 언어로 작성된 코드를 자동으로 생성하거나 오류를 수정해 주는 AI입니다.

예시: GitHub Copilot
GitHub Copilot은 개발자가 코드를 작성하는 동안 실시간으로 코드 추천을 해주는 AI 도구입니다. 코드 자동 완성뿐만 아니라, 함수와 클래스 작성도 도와줍니다.

Codex
Codex는 GPT-3 기반의 AI로, 자연어 명령을 프로그래밍 코드로 변환해 주는 기능을 가지고 있습니다. 사용자가 원하는 기능을 텍스트로 설명하면 이를 코드로 자동 변환할 수 있습니다.
